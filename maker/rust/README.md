# ip2region xdb rust 生成实现

## 使用方法
* 当前目录下执行 `cargo build --release` 编译生成工具  
* 目标生成在targe/release 目录下 maker  
* 使用方法: 
```
 Usage: maker --in-file <IN_FILE> --out-file <OUT_FILE>
 Options:
  -i, --in-file <IN_FILE>
  -o, --out-file <OUT_FILE>
  -h, --help                 Print help
  -V, --version              Print version
```
